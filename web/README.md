# Pildora Web App

Web application built with Next.js and TypeScript.

## Overview

Full-featured dashboard for medication management, reporting, vault management, and sharing.

- **Framework**: Next.js (SSR for landing/marketing, SPA for authenticated dashboard)
- **Crypto**: `pildora-crypto` compiled to WASM via wasm-bindgen
- **Auth**: SRP-6a (zero-knowledge authentication)

🚧 Not yet implemented. Planned for Phase 4 (Multi-Platform Expansion).
